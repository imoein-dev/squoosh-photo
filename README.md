# Squoosh Photo 📸

یک ابزار قدرتمند، سریع و کاملاً امن برای فشرده‌سازی و تغییر اندازه تصاویر به صورت محلی در مرورگر. این پروژه با الهام از ابزارهای معروفی مانند Google Squoosh ساخته شده است، اما با رابط کاربری مدرن‌تر و تجربه کاربری (UX) بهینه‌تر.

## ✨ ویژگی‌های کلیدی

* **پردازش کاملاً محلی (Client-Side):** تصاویر شما هرگز به هیچ سروری ارسال نمی‌شوند. تمام پردازش‌ها با استفاده از قدرت دستگاه شما و از طریق Web Workerها در مرورگر انجام می‌شود، که حریم خصوصی کامل شما را تضمین می‌کند.
* **پشتیبانی از فرمت‌های مدرن:** تبدیل و فشرده‌سازی به فرمت‌های `WebP`، `JPEG` و `PNG`.
* **مقایسه در لحظه (Before/After):** با استفاده از اسلایدر تعاملی، می‌توانید کیفیت تصویر اصلی را با نسخه فشرده‌شده به صورت زنده مقایسه کنید.
* **تغییر اندازه هوشمند:** امکان تغییر اندازه تصاویر با حفظ نسبت ابعاد (Aspect Ratio) و استفاده از الگوریتم‌های مختلف مانند `Lanczos3` یا کیفیت بالای مرورگر.
* **ابزارهای ویرایش:** قابلیت چرخش تصاویر، زوم (Zoom in/out)، جابجایی (Pan) و مشاهده تصاویر به صورت پیکسلی (Pixelated) یا نرم (Smooth).
* **رابط کاربری واکنش‌گرا (Responsive):** طراحی شده با Tailwind CSS برای اجرای روان در موبایل، تبلت و دسکتاپ.
* **تم تاریک و روشن:** پشتیبانی از تغییر تم پس‌زمینه برای بررسی بهتر لبه‌های تصاویر.

## 🛠 تکنولوژی‌های استفاده شده

* **فریم‌ورک:** [Next.js](https://nextjs.org/) (نسخه ۱۶) با React 19
* **استایل‌دهی:** [Tailwind CSS v4](https://tailwindcss.com/)
* **پردازش تصویر:** Canvas API و [browser-image-compression](https://www.npmjs.com/package/browser-image-compression) (اجرا شده در Web Worker برای جلوگیری از افت فریم)
* **تعاملات کاربری:**
  * `react-compare-slider` برای مقایسه قبل و بعد
  * `react-zoom-pan-pinch` برای قابلیت زوم و جابجایی
* **زبان:** TypeScript

## 🚀 راه‌اندازی و اجرا

برای اجرای این پروژه در سیستم خود، مراحل زیر را دنبال کنید:

### پیش‌نیازها
* نصب بودن [Node.js](https://nodejs.org/) (نسخه ۱۸ یا بالاتر)

### نصب

۱. مخزن را کلون کنید یا فایل‌ها را دانلود کنید:
```bash
git clone <repository-url>
cd squoosh-photo
```

۲. وابستگی‌ها را نصب کنید:
```bash
npm install
# یا اگر از yarn استفاده می‌کنید
yarn install
```

۳. سرور توسعه را اجرا کنید:
```bash
npm run dev
# یا
yarn dev
```

۴. مرورگر خود را باز کرده و به آدرس `http://localhost:3000` بروید.

## 💡 راهنمای استفاده

۱. **آپلود تصویر:** عکس مورد نظر خود را با کشیدن و رها کردن (Drag & Drop) یا کلیک روی قسمت مشخص شده وارد برنامه کنید.
۲. **تنظیمات فشرده‌سازی (سایدبار):**
   * در صورت نیاز ابعاد عکس را در بخش **Resize Image** تغییر دهید.
   * فرمت خروجی (WebP، JPEG، PNG) را انتخاب کنید.
   * کیفیت (Quality) را با استفاده از نوار لغزنده تنظیم کنید.
۳. **بررسی نتیجه:** تغییرات را با حرکت دادن خط مقایسه روی عکس مشاهده کنید. حجم جدید و میزان بهینه‌سازی در پایین سایدبار نمایش داده می‌شود.
۴. **دانلود:** در نهایت با کلیک روی دکمه **DOWNLOAD FILE** عکس بهینه‌شده را دریافت کنید.

## ⚙️ نکات عملکردی (Performance)

در این پروژه تلاش شده تا لگ و کندی به حداقل برسد:
* پردازش‌های سنگین به یک **Web Worker** مجزا (`public/workers/image-processor.js`) منتقل شده‌اند تا محیط کاربری (UI) هنگام فشرده‌سازی فریز نشود.
* از افکت‌های CSS (مانند backdrop-blur) به صورت بهینه استفاده شده است تا در سیستم‌های ضعیف‌تر و موبایل‌ها افت فریم رخ ندهد.
* فشرده‌سازی خودکار در زمان آپلود اولیه غیرفعال شده است تا کاربر بتواند بلافاصله تصویر را مشاهده کند و سپس تصمیم به اعمال تغییرات بگیرد.
